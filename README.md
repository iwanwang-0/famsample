# famsample

## Envirenment
Node version: v16.15.0  
Npm version: 8.5.5

## Add dependencies
``` shell
npm install
```

## Execution
Create a .env file, and fill into your private key.
``` shell
cp .env.example .env
```
Execute
``` shell
node index.js
```
